# Water Usage Tracker

## SDG 6: Clean Water and Sanitation

Water Usage Tracker is a mobile application developed using Kotlin and Android Studio to help users monitor, analyze, and improve their daily water consumption habits.

The application supports water usage tracking, statistical analysis, weather integration, location services, sensor integration, and community sharing. The project aligns with United Nations Sustainable Development Goal 6 (SDG 6) by promoting responsible water consumption and sustainable water management.

## Application Features

### Home Management

- Add multiple homes
- Track water consumption for each home
- Update water usage records
- Delete homes
- Undo water usage entries

### Statistics Screen

- View water usage statistics
- Monthly consumption visualization
- Progress monitoring
- Usage trend analysis

### Comparison Screen

- Compare water usage with benchmark standards
- Low benchmark
- Medium benchmark
- High benchmark

### Weather Information

- Retrieve real-time weather data
- Display temperature and weather conditions
- Weather API integration using Retrofit

### Location Services

- Detect user location automatically
- Display current city information
- Improve weather data accuracy

### Sensor Integration

- Access device step counter sensor
- Monitor user physical activity level
- Use activity data for hydration recommendations

### Smart Hydration Recommendation

The application provides personalized daily water intake recommendations based on:

- User body weight
- Current weather conditions
- Temperature data
- Physical activity level
- Step counter sensor data

### Community Sharing

- Share water-saving achievements
- Upload records to Firebase Firestore
- View community posts from other users
- Community-based motivation and awareness

### Profile Management

- Manage personal profile information
- Configure daily water targets
- Update hydration settings

### Reminder System

- Water intake reminder support
- Encourage regular hydration habits

## Database Implementation

### Firebase Firestore
