SDG 6: Clean Water and Sanitation

Water is one of the most important resources for sustainable development. Many households are unaware of their daily water consumption patterns, which can lead to unnecessary waste. Water Usage Tracker helps users monitor usage, analyse trends, receive personalised recommendations, and encourage responsible water conservation practices.

Projects

Project 1 – Water Usage Tracker

Project 1 Video GitHub Repository

Project 2 – Advanced Water Usage Tracker

Problem Statement: Effective water conservation requires accurate monitoring, environmental awareness and community participation. Traditional tracking systems rarely provide personalised recommendations or collaborative engagement.

Features:

  • Room Database
  • Firebase Firestore
  • Weather API using Retrofit
  • GPS Location Services
  • Step Counter Sensor
  • Community Sharing
  • Personalised Water Recommendation

Project 2 Video GitHub Repository

Lessons Learned: This project strengthened my understanding of MVVM architecture, cloud databases, REST APIs, mobile sensors and scalable Android application development.

Reflection & Learning Journey

I started this course by learning the basics of Jetpack Compose layouts and gradually progressed to state management, navigation, local databases and cloud services. Through Project 2, I successfully integrated Room Database, Firebase Firestore, Weather API, GPS location and step counter sensors into a complete Android application. This experience improved my technical skills and demonstrated how software engineering can contribute to SDG 6 by encouraging sustainable water management.
